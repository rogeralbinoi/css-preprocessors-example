# CSS Preprocessors

**Gulp**

        npm install --global gulp gulp-cli

**Tasks**

        gulp // inicia o gulp dev e o styl
        gulp dev // inicia watch nos arquivos styl
        gulp styl // gera os arquivos css a partir de arquivos stylus
